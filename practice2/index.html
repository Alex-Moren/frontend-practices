Estudia Desarrollo de Software en CETI

El Centro de Enseñanza Técnica Industrial (CETI) es una institución pública en Jalisco especializada en formación tecnológica. Sus estudiantes aprenden programación, electrónica, redes y desarrollo de software para resolver problemas reales usando tecnología.

¿Qué puedes aprender en CETI?


Programación

Aprendes a crear aplicaciones, páginas web y software.

Electrónica

Trabajas con circuitos y sistemas electrónicos.

Redes

Aprendes cómo funcionan las redes e internet.

Proyectos que puedes crear


Videojuegos

Programar juegos usando lógica y gráficos

Aplicaciones Web

Páginas interactivas usando HTML, CSS y Javascript

Robótica

Controlar robots con programación y electrónica


Tecnologías que aprenderás

HTML y CSS

Para crear páginas web.

JavaScript

Para hacer páginas interactivas.

Python / Java / C

Lenguajes para desarrollar software real.
